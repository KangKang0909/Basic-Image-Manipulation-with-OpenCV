# Basic Image Manipulation with OpenCV

This project implements fundamental **Image Manipulation** techniques using the **OpenCV** library. It serves as a foundational guide for performing essential operations required in computer vision workflows.

## 🚀 Overview
The goal of this project is to demonstrate the basic building blocks of image processing. By manipulating the underlying NumPy arrays that represent images, the application performs tasks such as color space conversion, pixel-level editing, and drawing.

## Key Features
* **Color Space Conversions:** Switching between **BGR**, **RGB**, **Grayscale**, and **HSV** formats.
* **Image I/O:** Reading, displaying, and saving images in various file formats.
* **Pixel Manipulation:** Accessing and modifying individual pixel values or regions of interest (ROI).
* **Drawing Functions:** Adding geometric shapes (lines, rectangles, circles) and text overlays to images.
* **Channel Splitting & Merging:** Isolating individual color channels (Blue, Green, Red) and recombining them.
